## 0. 版本约束

- [x] 版本定位冻结：`v1.1.2` 只做任务中心，不提前做 `v1.2` 素材库重构。
- [x] 关键规则冻结：

- [ ] 新增任务详情接口（浮层需要的播放/下载信息）。
