# Seedance v1.1.5 开发 TODO（分组 Key 路由）

发布日期：2026-05-08

## 0. 冻结目标

- [x] Seedance 只保留一个工具入口，不按 key 分叉 tool/skill。
- [x] API Key 选择统一在后端按用户组路由。
- [x] 本版本不做本地计费统计，费用在厂商平台查看。
- [x] 路由策略对后续 HappyHorse / GPT-Image-2 可复用。
- [x] 对 LLM 暴露稳定路由错误码，确保可执行的错误提示。

- [x] 新增 `config/key_routing.json` 样例与加载逻辑。
