# Seedance v1.1.5.1 接口契约（任务筛选与浮层字段增强）

发布日期：2026-05-09

> 说明：
> - 本文冻结 `v1.1.5.1` 新增/变更接口字段。
> - 前端任务页仅依赖本文定义的任务筛选与详情字段。

## 1. 总体约定

1. Base Path：`/api/v1/tasks`
2. 鉴权：`Authorization: Bearer <token>`
3. 时间字段：Unix 秒级时间戳（UTC）
4. 编码：`application/json; charset=utf-8`
5. 默认排序：`created_at desc, id desc`

## 2. 枚举约定

`TaskStatus`（沿用）：

1. `PENDING`
2. `RUNNING`
3. `SUCCEEDED`
4. `FAILED`
5. `CANCELED`

## 3. 接口变更

## 3.1 列表查询

`GET /api/v1/tasks`

新增 Query：

| 参数 | 类型 | 必填 | 默认 | 说明 |
|---|---|---|---|---|
| `group_id` | string | 否 | - | 用户组筛选（单值） |
| `start_at` | int | 否 | - | 创建时间起点（含） |
| `end_at` | int | 否 | - | 创建时间终点（含） |

保留 Query（节选）：

| 参数 | 类型 | 必填 | 默认 | 说明 |
|---|---|---|---|---|
| `user_id` | string | 否 | - | 用户筛选 |
| `provider` | string | 否 | - | 来源筛选 |
| `status` | string | 否 | - | 状态筛选 |
| `include_deleted` | bool | 否 | `false` | 包含软删除 |
| `offset` | int | 否 | `0` | 偏移 |
| `limit` | int | 否 | `48` | 页大小，最大 200 |

规则：

1. 时间过滤基于 `created_at`。
2. `start_at > end_at` 返回 `400`。
3. 未提供时间参数时不做时间过滤。

## 3.2 任务页组列表

`GET /api/v1/tasks/groups`

用途：任务页筛选下拉数据源。

权限：

1. 已登录用户可访问。
2. 返回全量组（非仅本人所属组）。

Response：

```json
{
  "groups": [
    {"group_id": "g1", "group_name": "国内投放"},
    {"group_id": "g2", "group_name": "海外投放"}
  ]
}
```

字段约定：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `group_id` | string | 是 | 组 ID |
| `group_name` | string | 是 | 组名称 |

## 3.3 任务详情/浮层字段增强

以下字段需要在列表项和详情项中可读（允许为 `null`）：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `prompt_text` | string/null | 否 | 提示词文本 |
| `generation_params` | object/null | 否 | 生成参数对象 |
| `prompt_resources` | array/null | 否 | 提示词资源映射 |

`prompt_resources` 子项：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | string | 是 | 资源名（如 `@Image.png`） |
| `url` | string | 是 | 可下载 URL |

## 4. 兼容约束

1. 历史任务可能无 `prompt_*` 字段，接口必须保持 200 并返回 `null/[]`。
2. 新字段缺失时前端不得报错，显示“暂无”。

## 5. 错误契约（新增部分）

新增参数错误建议：

```json
{
  "detail": {
    "code": "INVALID_TIME_RANGE",
    "message": "start_at must be <= end_at"
  }
}
```

建议 HTTP 状态码：

1. `400` 参数非法（如 `start_at > end_at`）
2. `401` 未登录
3. `403` 无权限
4. `500` 服务内部错误

## 6. 前端 URL 状态建议（非接口）

任务页建议使用以下 query key 持久化状态：

1. `user_id`
2. `group_id`
3. `provider`
4. `status`
5. `include_deleted`
6. `time_preset`
7. `start_at`
8. `end_at`

## 7. 前端复制行为约束（非接口）

1. 浮层“复制提示词和参数”按钮输出仅可读文本。
2. 不输出 JSON 片段。
