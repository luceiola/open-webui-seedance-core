# Seedance v1.1.2 接口契约（任务中心）

发布日期：2026-05-07

> 说明：
> - 本文为 `v1.1.2` 任务中心冻结草案。
> - 对应能力：全量任务列表、按用户筛选、自动归档、缩略图预览、本地下载、软删除。

## 1. 总体约定

- Base Path：`/api/v1/generation-tasks`
- 鉴权：`Authorization: Bearer <token>`
- 可见性：全员可见全量任务
- 删除权限：创建者或管理员
- 时间字段：Unix 秒级时间戳

前端展示约束（冻结）：

1. 任务面板使用网格模式，按视口宽度自适应 3-8 列。
2. 任务卡片宽高比固定 `3:4`。
3. 列表卡片与浮层均提供下载图标。
4. 浮层删除图标以 `can_delete` 字段为准。

错误响应（统一）：

```json
{
  "ok": false,
  "error_code": "Forbidden",
  "error_message": "No permission to delete this task",
  "request_id": "..."
}
```

## 3. 任务列表

### 3.1 查询任务列表

`GET /api/v1/generation-tasks`

Query：

- `user_id`（可选）
- `status`（可选）
- `model`（可选）
- `limit`（默认 50，最大 200）
- `offset`（默认 0）
- `include_deleted`（默认 false）

Response：

```json
{
  "ok": true,
  "total": 1234,
  "items": [
    {
      "task_id": "cgt-xxx",
      "user_id": "user_1",

      "model": "doubao-seedance-2-0-260128",
      "status": "RUNNING",
      "archive_status": "NOT_REQUIRED",
      "download_ready": false,
      "can_delete": false,
      "thumbnail_url": null,
```

## 4. 任务详情与预览

### 4.1 查询任务详情

`GET /api/v1/generation-tasks/{task_id}`

Response：`TaskItem`

### 4.2 获取任务预览信息

`GET /api/v1/generation-tasks/{task_id}/preview`

## 5. 下载与归档

### 5.1 下载到本地（用户动作）

`GET /api/v1/generation-tasks/{task_id}/download`

规则：

1. 必须从平台归档文件返回。
2. 未归档成功返回 409（`ArchiveNotReady`）。
3. 以流式下载返回（`Content-Disposition: attachment`）。

### 5.2 归档重试（系统/管理员）

`POST /api/v1/generation-tasks/{task_id}/archive/retry`

Response：

```json
{
  "ok": true,
  "task_id": "cgt-xxx",
  "archive_status": "PENDING"
}
```

## 6. 删除

### 6.1 软删除任务

`DELETE /api/v1/generation-tasks/{task_id}`

规则：

1. 仅创建者或管理员可删。
2. 实际执行软删除，写入 `deleted_at/deleted_by`。
3. 默认列表不返回软删除记录。

Response：

```json
{
  "ok": true,
  "task_id": "cgt-xxx",
  "deleted_at": 1777000400
}
```

## 7. 后台任务与保留期

### 7.1 自动归档触发

任务状态进入 `SUCCEEDED` 后自动触发归档：

1. 拉取上游视频 URL
2. 下载到平台存储
3. 写入 `archive_status`
4. 生成缩略图（失败则占位）

### 7.2 保留期清理

建议配置：

- `TASK_SOFT_DELETE_RETENTION_DAYS=7`

清理器行为：

1. 清理超保留期软删除任务元数据
2. 清理关联归档文件与缩略图文件

## 8. 状态码建议

1. `400` 参数非法
2. `403` 无权限
3. `404` 任务不存在
4. `409` 归档未完成，暂不可下载
5. `500` 服务内部错误
