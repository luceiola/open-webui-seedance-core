# Seedance v1.1.5.2 需求文档（修正版）

发布日期：2026-05-13

## 1. 版本定位

`v1.1.5.2` 为 `v1.1.5.1` 的增量修正版，目标是修复线上/联调中发现的体验与一致性问题，不引入大范围架构改动。

一句话：**小步快修，保持稳定。**

## 2. 修正范围

1. 视频预览弹窗“复制”按钮无效问题修复。
2. 任务失败后在弹窗展示失败原因，优先显示 API 返回的错误信息。
3. 筛选改为“选择即应用”，移除“筛选/查询”按钮。
4. 时间筛选移除“自定义时间”，新增“最近3个月”“最近半年”。
5. 移除“包含已删除”筛选项。
6. 刷新按钮调整到筛选同一行，改为正方形并与筛选框同高。

## 3. 目标

1. 修复已确认问题并保持现有功能不回退。
2. 保持 dev/prod 发布流程与隔离策略不变。
3. 补齐最小必要测试与回归验证。

## 4. 非目标

1. 不引入 `v1.2` 范围能力。
2. 不做无关视觉重构或新功能扩展。
3. 不调整现有 key routing 与环境隔离主方案。

## 5. 验收标准（DoD）

1. 弹窗复制按钮在常规浏览器环境可用，失败时有明确错误提示。
2. `FAILED` 任务在弹窗展示错误原因，优先顺序：
1. `error_message`
2. `error_code`
3. `archive_error`
3. 修改任一筛选项后 200ms 内自动刷新结果，无“筛选”按钮。
4. 时间筛选仅保留：全部、24小时、7天、30天、3个月、半年。
5. 页面不再出现“包含已删除”选项。
6. 刷新按钮位于筛选区同列，视觉尺寸为正方形且高度与筛选框一致。
7. 关键主路径回归通过（任务创建、任务列表、任务详情/浮层、下载）。
